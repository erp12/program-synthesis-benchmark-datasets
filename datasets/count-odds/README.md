# Count Odds

## Edge Cases
32 cases
- 1 - empty vector
- 25 - length 1 vectors
- 6 - length 2 vectors

## Random Cases
1 million cases
- 100,000 vectors of random length, all odd integers
- 100,000 vectors of random length, all even integers
- 800,000 vectors of random length, random integers

Train cases (200)

Test cases (2,000)
